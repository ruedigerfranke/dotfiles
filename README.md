# Dotfiles

My dotfiles managed by [rcm](https://github.com/thoughtbot/rcm).

## Get started

1. Install [rcm](https://github.com/thoughtbot/rcm)
```
brew install rcm
```

2. Clone dotfiles repository
```
git clone git@github.com:ruedigerfranke/dotfiles.git ~/.dotfiles
```

3. Run rcup
```
rcup
```
